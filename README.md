# 💼 Sistema de Conta Bancária em Java

Aplicação desenvolvida em Java simulando operações básicas de uma conta bancária.

## 📌 Funcionalidades
- Depósito
- Saque
- Consulta de saldo
- Menu interativo

## 🛠 Conceitos aplicados
- Programação Orientada a Objetos (POO)
- Encapsulamento
- Métodos
- Classes
- Controle de fluxo

## 🎯 Objetivo
Projeto criado para praticar fundamentos de orientação a objetos e estruturação de aplicações em Java.
